# UI 스타일링 표준 (UI/Styling Standard)

## 1. 기본 원칙
- **Tailwind CSS 4**: 모든 스타일링은 Tailwind CSS를 최우선으로 사용합니다.
- **Theme 기반 설계**: `index.css`의 `@theme` 섹션에 정의된 변수를 사용합니다.
- **이모지 & 이미지 하이브리드**: `AssetProvider`를 통해 이모지 모드와 이미지 모드를 실시간으로 전환할 수 있도록 구현합니다.

## 2. 애니메이션 규칙
- **빙고 플래시**: 빙고 달성 시 `@keyframes flash`를 사용한 시각 효과 부여.
- **카드 발동**: 순차적 발동 시 해당 카드에 `pulse` 효과 부여.

## 3. 레이아웃
- 3단 분할 레이아웃 (좌: 플레이어, 중: 그리드, 우: 적)을 기본으로 하며, 반응형 대응을 유지합니다.
